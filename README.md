
# Smart Cricket Team Selector

A console-based Core Java project that recommends cricket players based on match conditions such as target score, balls remaining, and wickets left.

## Features

- Player recommendation system
- Match condition analysis
- Required run rate calculation
- OOP Concepts implementation
- Dynamic team selection logic

## Technologies Used

- Core Java
- OOP Concepts
- ArrayList
- Conditional Statements
- Scanner Class

## Project Logic

The system analyzes:
- Required run rate
- Strike rate of batsmen
- Economy rate of bowlers
- Match pressure situations

Based on these conditions, the system recommends suitable players.

## Sample Input

Enter Target Score: 120  
Enter Balls Remaining: 60  
Enter Wickets Left: 4

## Sample Output

Required Run Rate: 12.0

Recommended Players:
Virat Kohli -> Power Hitter
Rohit Sharma -> Power Hitter
